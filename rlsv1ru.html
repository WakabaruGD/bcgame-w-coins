Правила игры "Монетный Трон"

Монетный Трон - это игра с использованием монеток для 2-3 игроков.

Цель игры

Набрать >6 очков.
Решка дает 2 очка, Орел - 1 очко.
Очки даются за Повышенные монеты, которые зарабатываются в ходе игры.

Компоненты

  • По 7 монет на игрока
  • 1 маркер Лидера (правила игры)

Подготовка

  • Все игроки кладут 7 монет в 1 стопку орлом вверх - это Стопка Орлов
  • Оставьте место для будущей Стопки Решек и Зоны Повышенных монет
  • Игрок, предложивший сыграть, становится Лидером (берет правила игры)
РАУНД:

1. Скрытая ставка

  • Игрок берет монету, лежащую орлом вверх, из Стопки Орлов
  • Если после взятия монеты стопка пуста: верните верхнюю монету из Стопки Решек в Стопку Орлов решкой вверх. Затем верните остальные монеты из Стопки Решек орлом вверх.
  • Если нельзя взять монету (есть только 1 монета, лежащая решкой вверх): начинается ФИНАЛ.
  • Игрок скрытно выбирает сторону взятой монеты (О/Р) в качестве ставки.

2. Раскрытие

Все одновременно открывают свои ставки.

3. Результат

Согласие (все ставки одинаковы):

  • Маркер Лидера передается следующему игроку по часовой стрелке
  • Новый Лидер Повышает ставку орлом вверх
  • Все возвращают ставки в Стопку Решек решкой вверх (если она пуста - формируют Стопку Решек)

Несогласие (ставки разные): по очереди, начиная с Лидера и далее по часовой стрелке, игроки применяют эффект своей ставки:

  • Любая ставка Лидера: Лидер повышает ставку той стороной вверх, которой поставил
  • Орел (кроме Лидера): игрок забирает маркер Лидера и Повышает ставку орлом вверх
  • Решка (кроме Лидера):
    • Игрок берет верхнюю монету из Стопки Орлов
    • Кладет в Стопку Решек решкой вверх
    • Возвращает ставку решкой вверх

4. Трансформация

Если у игрока в Зоне Повышенных монет орлом вверх лежит 2 монеты, он должен:

  • Перевернуть одну из этих монет решкой вверх
  • Другую отправить в Стопку Решек решкой вверх

5. Проверка победы

Побеждает любой игрок, чья сумма очков за Повышенные монеты >6 и больше, чем у других. При равенстве - играем дальше с текущим Лидером.

ФИНАЛ

  • Игрок, у которого раньше всех закончились монеты для ставок, становится Лидером
  • Игроки могут использовать для ставки любую монету из любой стопки (но не из Зоны Повышенных монет)
  • Игроки делятся на команды: Узурпатор (только Лидер) vs Претенденты (остальные игроки)
  • Ставка: Претенденты тайно договариваются об одной ставке. Все делают скрытую ставку, раскрывают одновременно
  • Результат:
    • Лидер побеждает при любых РАЗНЫХ ставках
    • Аутсайдеры побеждают при любых ОДИНАКОВЫХ ставках
  • Каждый в победившей команде Повышает свою ставку решкой вверх
  • Итог: Суммируйте все Повышенные монеты (О=1, Р=2). Побеждает игрок с самой большой суммой. При равенстве побеждает Лидер или ближайший к получению маркера Лидера.

Ключевые термины

  • Повысить: отложить ставку в Зону Повышенных монет (дают очки)
  • Лидер: Игрок с маркером Лидера (правилами)
  • Стопка Орлов: отсюда берутся монеты для ставки (можно брать только монеты орлом вверх)
  • Стопка Решек: сюда возвращаются монеты решкой вверх, если ставку не Повысили
  • Финал: раунд с особыми правилами, завершающий игру
